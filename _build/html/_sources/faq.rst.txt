Frequently asked questions
==========================


How do i change name of a table?
--------------------------------------------------------------

In order to change the table name you will have to manually change it using
sql `rename` command and then after change the `table_alias` or `table_name`
depending if its table has changed in the source or you just want to change the table alias.
