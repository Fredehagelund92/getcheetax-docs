Cheetax - Exasol data loaderÂ¶

Cheetax is a command-line interface to load data from various sources into Exasol. Its primary audiance are data science that would like to get started quickly with a fully functional datawarehouse. |

For maximum performance and compatibility the commands are built upon native Exasol functionality like the import statement and UDF. This ensures that everything is running in the acriss the cluster. to learn more about how UDF and Import statement, please check out the Exasol documentation.

Cheetax is free to use (MIT license), open source (GitHub), available for Linux, OSX, and Windows.

Cheetax is routinely tested with loading data from JDBC (MySQL, PostgreSQL, EXASOL, , MSSQL, Oracle), CSV and JSON. However it will probably work with other JDBC connectors aswell.
